1Download & Prepare Plugin

  1. Create a new folder on your computer named com.cuer.showcaller.streamDeckPlugin
  2. Inside this folder, create these subfolders: js, css, and Images
  3. Download each file from the links above into the correct subfolder:
    • Core files ‚Üí main folder
    • JavaScript files ‚Üí js folder
    • CSS files ‚Üí css folder
    • Image files ‚Üí Images folder

2Install the Plugin

  1. Double-click the com.cuer.showcaller.streamDeckPlugin folder
  2. Stream Deck software will automatically install it
  3. The plugin will appear in your actions list

3Login to Your Account

  1. Drag any Cuer action to your Stream Deck
  2. In the property inspector, click "Login to Cuer"
  3. A popup window will open
  4. Sign in with your Cuer account credentials
  5. The popup will close automatically when successful

4Configure Buttons

  1. After logging in, click "Refresh Rundowns"
  2. Select the rundown you want to control
  3. Repeat this process for additional buttons
  4. Each button can control a different rundown if needed

5Start Controlling

  1. Press the Play/Pause button to start your rundown
  2. Use Forward/Backward to navigate segments
  3. Add a Status Display to monitor progress
  4. Press Reset to jump back to the beginning
Important: Each button needs to be configured by logging in and selecting a rundown. The plugin communicates with Cuer's servers, so an internet connection is required.
